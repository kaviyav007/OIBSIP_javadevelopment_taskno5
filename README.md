# Library Management System

## Project Description

The Library Management System is a Java-based desktop application developed to automate and simplify the daily operations of a library. The system provides an efficient way to manage books, users, book issuing, book returns, and library records through a user-friendly graphical interface.

The project is designed with separate functionalities for administrators and users. Administrators can manage book records and maintain the library database, while users can browse, search, issue, and return books conveniently.

## Features

### Admin Module

* Add new books to the library
* Manage library records
* View available and issued books
* Maintain book inventory

### User Module

* Browse available books
* Search books by title
* Issue books
* Return books
* View book availability status

### Additional Features

* Login Authentication
* Graphical User Interface (GUI) using Java Swing
* Real-time book status tracking
* Easy-to-use navigation and controls

## Technologies Used

* Java
* Java Swing
* Object-Oriented Programming (OOP)
* Collections Framework (ArrayList)

## Objectives

* Automate library management operations
* Reduce manual record-keeping
* Improve efficiency in book management
* Provide a simple and interactive user experience

## Outcome

This project demonstrates the practical implementation of Java programming concepts, GUI development, event handling, and object-oriented design principles while providing an effective solution for managing library activities.
